Features and Updates

Updates

August 1 7:38am - Added Single Scrims. Users can now post ads for singles scrims. The leader does not have to be part of team. Added lots of small bug fixes and improvements.

July 27 4:27pm - Users can now edit their profile and choose: their skill level / division, main class, if they want to be listed as mercs/ringers. Searching for mercs/ringers is done in the user page. We will be adding an option to select multiple classes and multiple divisions for a user.

July 26 8:31am - added team types (4v4,6v6,9v9). To set a type for your existing teams, go to the edit team page and choose a type. All new teams will have the option to choose a type. Filters will be updated soon.

July 26 6:55am - updated time zones based on feedback. Time zones are now represented using UTC offsets. You can edit your team's time zone by clicking the "Edit" button on the team's page.
